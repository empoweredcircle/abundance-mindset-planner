# Abundance Mindset Daily Planner

## Deploy to Vercel — No build step needed

### Step 1 — Upload to GitHub
1. Go to github.com → sign in
2. Click + → New repository
3. Name it `abundance-mindset-planner`, select Public, click Create
4. Click "uploading an existing file"
5. Drag BOTH files (index.html and vercel.json) into the upload area
6. Click Commit changes

### Step 2 — Deploy on Vercel
1. Go to vercel.com → sign in with GitHub
2. Click Add New Project → import abundance-mindset-planner
3. When asked about Framework Preset — select **Other**
4. Leave everything else as default → click Deploy
5. Your live URL appears in about 30 seconds

That's it. There is no build process — Vercel simply serves the HTML file directly.
